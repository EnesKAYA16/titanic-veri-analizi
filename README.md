# 🚢 Titanic Veri Analizi

Titanic yolcu verisi üzerinde keşifsel veri analizi (EDA) projesi.

## 📊 Bulgular
- Kadınların hayatta kalma oranı erkeklere göre çok daha yüksek
- 1. sınıf yolcuları 3. sınıfa göre çok daha fazla hayatta kaldı
- Genç yolcuların hayatta kalma oranı daha yüksek

## 🛠️ Kullanılan Teknolojiler
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dosyalar
- `analiz.ipynb` → Tüm analiz ve grafikler
